# Fraud-Detection-Risk-Analysis-System
## Project Overview
- This project focuses on detecting fraudulent transactions and analyzing risk patterns using data analytics techniques. The goal is to identify fraud trends, measure risk exposure, and provide actionable business insights.
- The project demonstrates end-to-end data analytics skills including data cleaning, SQL analysis, and interactive dashboard development.

## Objectives
- Identify fraudulent transactions
- Analyze fraud patterns by device, location, and category
- Measure fraud rate and financial risk
- Provide business insights for fraud prevention
- Build an interactive dashboard for monitoring

## Tools & Technologies
- **Python** (Pandas, NumPy, Matplotlib, Seaborn)
- **SQL** (Data querying & aggregation)
- **Power BI** (Dashboard & visualization)
- **Jupyter Notebook**

## Key KPIs
- Total Transactions
- Fraud Rate %
- Fraud Cases
- Amount at Risk
- Monthly Fraud Trends
- Fraud by Location & Device Type 

## Dashboard Insights
- Fraud rate analysis over time
- Device-wise fraud distribution
- Location-based fraud comparison
- Category-wise fraud trends
- Weekday vs Weekend fraud patterns

## Business Insights
- Certain locations show higher fraud frequency
- Fraud rates fluctuate monthly indicating seasonal patterns
- Specific transaction categories have higher fraud risk
- Fraud occurrences vary across devices and security methods

## Outcome
This project helps businesses:
- Detect fraud early
- Reduce financial risk
- Improve transaction security
- Make data-driven fraud prevention strategies
